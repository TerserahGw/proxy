# Proxy Auto Checker

Upload semua isi folder ini ke repository GitHub kamu.

Fitur:
- Cek source proxy luar setiap 10 menit.
- Tambahkan proxy baru yang valid ke `verified-proxies.txt` dan `verified-proxies.json`.
- Cek ulang repo setiap 1 jam.
- Hapus proxy yang sudah tidak valid.
- Source milik repo kamu sendiri tidak dipakai sebagai input, hanya sebagai output.

Setelah upload, aktifkan:

Settings → Actions → General → Workflow permissions → Read and write permissions
